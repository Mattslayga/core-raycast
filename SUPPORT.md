# Support

Core Edge for Raycast is currently a public OSS dev-mode beta for technical
users. It is not yet distributed through the Raycast Store.

## Where to Ask

- Use GitHub issues for reproducible bugs and focused feature requests.
- Use private security advisories for vulnerabilities or token-handling issues.
- Do not use public issues for secrets, private Core data, or exploit details.

## What to Include

For bugs, include:

- command name,
- expected behavior,
- actual behavior,
- reproduction steps using sanitized data,
- Core Edge server version if known,
- token role only (`read`, `write`, or `unknown`).

Never include:

- API tokens,
- private Core record content,
- raw server responses containing private data,
- screenshots of Raycast preferences,
- screenshots with private record titles, tags, IDs, or content.

## Token Exposure

If a token appears in an issue, pull request, screenshot, log, chat, or shared
terminal output, revoke it before continuing the report.
